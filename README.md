# Gear-to-Go

A simple kayaking gear rental cart prototype built to explore state management, input validation, and dynamic UI updates using JavaScript.

## Demo
https://gear-to-go.netlify.app/

## Features
- Add rental items to a cart with size selection
- Choose rental length with validation
- Real-time total cost calculation
- Remove items from the cart
- Dynamic UI updates using event delegation

## Tech
- HTML
- CSS
- JavaScript

## What I Learned
- Managing application state using arrays and objects  
- Using event delegation to handle dynamic elements  
- Validating user input and handling edge cases  
- Updating the DOM efficiently based on state changes
